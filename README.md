# HETGNN-FR: Fraud Ring Detection Using Heterogeneous Graph Neural Networks

**Author:** Milind Sahu  
**Institution:** Indian Institute of Science Education and Research (IISER) Tirupati

## Overview

HETGNN-FR is a research project that detects coordinated financial fraud rings using
Heterogeneous Graph Neural Networks (HGNNs). The system models accounts, devices,
merchants, and IP addresses as a heterogeneous graph and applies a Heterogeneous Graph
Transformer (HGT) combined with temporal encoding to identify suspicious groups of
accounts acting together.

### Key Features

- Heterogeneous Graph Transformer (HGT)
- Time2Vec temporal encoding
- Fraud ring extraction via sliding-window analysis
- Ring-level classification using learned graph embeddings
- Explainability module for analyst investigation
- Interactive Streamlit dashboard
