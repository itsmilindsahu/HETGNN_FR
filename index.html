Key Contributions

🧠 Heterogeneous GNN

Models accounts, devices, merchants, and IP addresses within a unified graph structure.

⏳ Temporal Encoding

Time2Vec representations capture temporal clustering patterns frequently observed in coordinated fraud.

🔍 Ring Discovery

Sliding-window graph extraction discovers hidden fraud rings instead of isolated suspicious transactions.

📊 Explainability

Provides interpretable signals through shared infrastructure analysis and suspicious edge scoring.
